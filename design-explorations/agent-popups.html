💡 Key Design Decisions

  • V1 (Recommended) — The sidebar+main layout is the best balance. It keeps stat bars always visible while the main area uses tabs to switch between department panels. The 8-cell asset preview grid gives a quick visual of recent work. The queue mini-cards add the missing Kanban visibility.
  • V2 — Best for widescreen desktops. Shows ALL info at once. The 5-column Kanban mini board is the standout feature — lets you see the full pipeline from any agent popup.
  • V3 (Quick win) — Minimal code change. Replaces the current cramped layout with a terminal-style log view. Easy to implement since it mostly reuses existing log data.
