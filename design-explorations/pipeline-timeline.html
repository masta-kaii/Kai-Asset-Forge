📊 Timeline Comparison & Recommendations

  • T1 — Step Pipeline Bar is the highest-impact quick win. Replace the current PipelineBar component with this multi-step design. Each step shows an index number, progress bar, and agent icon. The arrow connectors between steps add game-like polish. Can be implemented in ~30 minutes.
  • T2 — Dungeon Map Vertical is best as a side drawer or in the Popo command center. Adds descriptions, ETA, and inline progress bars. Great for detailed pipeline monitoring but takes more screen space. ~1-2 hours to implement.
  • T3 — Cycle History Gantt adds a completely new dimension — historical data visualization. Each column is one cycle, each cell is a pipeline step (green=done, gold=active, red=failed). Best feature: shows failure tracking and throughput at a glance. ~2-3 hours to implement (needs cycle history state).
  • Combined concept: The compact HUD + timeline bar at the bottom of the dungeon floor creates a complete game-like HUD system. The timeline replaces the current bare-bones PipelineBar.
